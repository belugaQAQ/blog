# Hexo主题使用教程

本文介绍如何使用Material主题的分类功能。

## 什么是分类

分类是组织文章的一种方式，可以帮助读者快速找到相关主题的文章。与标签不同，分类通常是有层级关系的。

## 如何创建分类

在文章的Front Matter中添加`categories`字段：

```yaml
categories: 教程
```

或者创建嵌套分类：

```yaml
categories:
  - 教程
  - Hexo
```

## 分类页面

主题会自动生成分类页面，显示所有分类及其文章数量。访问`/categories/`可以查看所有分类。

## 分类与标签的区别

- 分类：有层级关系，数量较少，用于组织文章
- 标签：无层级关系，数量较多，用于标记文章内容

## 最佳实践

1. 每篇文章只设置1-2个分类
2. 分类名称应该简洁明了
3. 保持分类结构的一致性
4. 定期整理分类，避免分类过多过杂

## 总结

合理使用分类功能可以让你的博客更加有序，方便读者浏览和查找内容。
